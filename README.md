# Customer Sales Analysis 📊

## Overview
Analysis of Superstore sales data using Python, Pandas, SQL and data visualization libraries.

## Tools Used
- Python
- Pandas
- Matplotlib & Seaborn
- SQLite (SQL)
- Jupyter Notebook

## Key Insights
1. West region has highest sales (~$725,000)
2. Technology is most profitable category
3. Sales peak every November-December
4. Furniture has high sales but low profit
5. Canon imageCLASS is most profitable product

## Project Structure
- data/ — raw dataset
- notebooks/ — Jupyter analysis notebook
- outputs/ — saved chart images
